Privacy Policy

Last Updated: January 23, 2026

Introduction

This privacy policy explains how the Google Sheets Unpivot Extension ("the Extension") collects, uses, and protects user data.

Data Collection

The Extension collects the following data:

  • OAuth 2.0 Access Tokens: Used to authenticate with Google Sheets API to read and write spreadsheet data.

Data Usage

Authentication tokens are used exclusively for:

  • Reading data from the selected range in your Google Sheets
  • Creating new sheets to display unpivoted results
  • Writing the transformed data to your spreadsheet

Data Storage

OAuth tokens are stored locally in your browser using Chrome's storage API. No data is transmitted to external servers owned or operated by the extension developer.

Data Sharing

We do not:

  • Share user data with third parties
  • Sell user data
  • Use user data for advertising or marketing
  • Transfer user data outside of Google's official APIs

Data Security

All authentication is handled through Google's secure OAuth 2.0 system. The Extension follows Chrome Web Store security best practices and does not execute remote code.

User Rights

You can:

  • Revoke the Extension's access at any time through your Google Account settings
  • Uninstall the Extension to remove all locally stored data
  • Request information about your data by contacting us

Changes to This Policy

We may update this privacy policy from time to time. Changes will be posted on this page with an updated "Last Updated" date.
